# PandaDoc Contracts - WCC Student Enrollment

PandaDoc Contract Libraries and Templates for WCC student enrollment document signing.

## Structure

```
├── templates/           # PandaDoc document templates (JSON definitions)
├── libraries/           # Reusable content library items (clauses, sections, fields)
├── schemas/             # Field schemas and variable definitions
├── examples/            # Example filled contracts for reference
└── docs/                # Process documentation and setup guides
```

## Workflow

1. **Content Library** - Reusable clauses, terms, and sections stored in `libraries/`
2. **Templates** - Full contract templates built from library items in `templates/`
3. **Schemas** - Variable/field definitions for dynamic data population in `schemas/`
4. **Signing** - Templates configured with PandaDoc signing workflows

## Template Types

- **Student Enrollment Agreement** - Primary enrollment contract for new students
