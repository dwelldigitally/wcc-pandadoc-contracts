# PandaDoc Branding & Design

## Logos
- Upload PNG, JPG, or GIF (scaled to 200w x 50h)
- Import from Dropbox, Google Drive, Box, OneDrive
- Place in headers, footers, or anywhere in document body

## Colors
- Set theme colors for:
  - Headings
  - Hyperlinks
  - "Open the document" button in emails

## Fonts
- 6 built-in fonts
- Custom fonts from Google Fonts

## Document Themes
- Create reusable design themes:
  - Fonts
  - Colors
  - Paper size
  - Headers/footers
- Apply theme to any template

## Headers & Footers
- Consistent logos across all pages
- Page numbers
- Contact details
- Institution name and address

## Email Customization
- Custom footer text
- Button colors
- Subject line templates

## Canva Integration
- Access Canva design features directly within PandaDoc
- For more advanced visual customization

## Enrollment Branding Setup

1. Create a document theme with institution's logo, colors, and fonts
2. Apply to ALL enrollment templates for consistent look
3. Customize email templates with admissions department branding
4. Include institution seal/logo in header
5. Use school colors for headings and accent elements
