# PandaDoc Bulk Send

## Overview

Send the same template to many recipients at once — critical for enrollment at scale.

## How It Works

1. Select a template
2. Upload a CSV file with recipient data (name, email, program, start date, etc.)
3. PandaDoc generates and sends up to **1,000 personalized documents per hour**

## Personalization

- Template variables auto-populate from CSV columns
- Each student gets a unique document with their specific data
- Supports all field types (text, dates, dropdowns)

## Requirements

- Template must have at least one role (e.g., "Student")
- CSV cannot contain duplicate email addresses
- Available on **Business and Enterprise plans** with volume package add-on

## Credits

- Business Annual and Enterprise: 120 usage credits/year
- Each bulk-sent document = 1 credit
- Additional credits purchasable

## Document Bundling

Send multiple related documents as a single bundle:
- Enrollment Agreement
- Financial Disclosure
- Housing Contract
- FERPA Release

Students complete everything in one signing session.

## Enrollment Workflow

1. Export student list from SIS/CRM as CSV
2. Map CSV columns to template variables
3. Bulk-send all enrollment contracts for a cohort
4. Bundle related forms together
5. Monitor completion via dashboard
6. Auto-remind non-signers
