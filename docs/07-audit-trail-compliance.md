# PandaDoc Audit Trail & Compliance

## Audit Trail

### Tracked Events
- Document creation, sending, viewing, signing
- Approval, rejection, decline
- Expiration
- Recipient added/removed/reassigned
- All edits
- Payment events
- Verification attempts

### Metadata Captured
- Timestamps
- IP addresses
- User identity
- Location

### Access
- Open any sent document → click "Audit trail" (top right)
- API: List Document Audit Trail endpoint
- Admin: Workspace-level audit log of all user and document activity

## Certificate of Completion

Automatically appended as the **last page** of downloaded PDF:
- Document reference number
- Signer names
- Verified email addresses
- IP addresses
- Locations
- Timestamps (sent/viewed/completed)

## eSignature Compliance

### US Compliance
| Standard | Description |
|----------|-------------|
| **ESIGN Act** | Electronic Signatures in Global and National Commerce Act |
| **UETA** | Uniform Electronic Transactions Act (47 states + DC) |
| **FERPA** | Family Educational Rights and Privacy Act — **critical for education** |

### EU/International Compliance
| Standard | Description |
|----------|-------------|
| **eIDAS** | All 3 levels: SES, AES, QES |
| **GDPR** | Data protection compliance |

### Security
| Standard | Description |
|----------|-------------|
| **SOC 2 Type II** | Security and availability controls |

## Signature Types Supported
- Typed signatures
- Drawn signatures
- Uploaded signature images
- Digital signature certificates (cryptographic verification)

## Enrollment Compliance Notes

- **FERPA compliance** means PandaDoc can legally handle student records
- Audit trail + signature certificate = legally defensible proof of enrollment agreement signing
- Critical for: financial aid audits, accreditation reviews, dispute resolution
- Certificate automatically embedded in completed PDF — no extra steps needed
